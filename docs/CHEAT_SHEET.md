# Go Cheat Sheet

- `go version` — show Go version
- `go env` — environment info
- `go run file.go` — run a program
- `go build` — build binary
- `go test ./...` — run tests
- `go fmt ./...` — format code
- `go mod init` / `go mod tidy` — modules
- `go get pkg@version` — add dependency
- `go vet ./...` — static checks

Short snippet: `for i := 0; i < n; i++ {}`
